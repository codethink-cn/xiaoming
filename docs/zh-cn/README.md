# 小明 - 通用通讯软件机器人框架

小明是一个通用的、高效率的、易于上手的通用通讯软件机器人框架。目前，`QQ` 部分基于 [Mirai](https://github.com/mamoe/mirai) （一款杰出的高效率 `QQ` 机器人框架）实现，未来小明还将支持其他通讯软件。
